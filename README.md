# Cloud Test Application (CST-323)

Spring Boot + MySQL cloud test application for Activity 1.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL (Azure MySQL)
- Bootstrap

## Features
- Customers: Create, Read, Update, Delete
- Products: Create, Read, Update, Delete
- Orders + Order Items: Create, Read, Delete (with Customer + Product relationships)

## Run Locally
```bash
./mvnw spring-boot:run
